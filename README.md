📝 Registration Form

A responsive and accessible registration form built using HTML5 and CSS3.

🚀 Features

Proper label and input linking using for and id

Accessible structure with fieldset and legend

Client-side validation using:

required attributes

pattern for password rules

min and max for age input

Responsive layout using vw, max-width, and min-width

Clean dark-themed UI

🛠 Technologies Used

HTML5

CSS3

📚 What I Learned

Structuring forms using semantic HTML

Writing accessible and clean markup

Grouping CSS selectors properly

Avoiding duplicate CSS declarations

Debugging and passing structured test cases

Basic responsive design principles
